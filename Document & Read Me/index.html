B) Features - top

  1. Supports iOS & Android including iPad & Tablets and tested on all major devices .
  2. Whats app clone with PhoneGap / Cordova Full Hybrid App using AngularJS + Ionic framework which is open source front-end SDK for developing hybrid mobile apps with HTML5.
  3. Login with Facebook using Facebook plugin.
  4. Support all countries and phone number validation’s implemented for every country (Example US based phone number are 10 digit where as Australia’s phone number are 9 digit etc.)
  5. Contact syncing same as What’s App where we validate all phone book contacts with the one who has installed whats app ( against node.js API layer) and option to chat with them.
  6. Direct socket chat with private messaging feature similar to whats app .
  7. Added “Typing…” Features where a user get notified when other user is typing a message to them.
  8. Shipped with entire Socket.io backend layers with required connection handling and handshaking mechanism.
  9. Integrated with Node.JS JSON Api coded using sailjs.org framework and robust Mongo database integrated hence easy to scale if app goes to millions of users.
  10. Customized Material Design based Edit Profile screen and option to change avatars and status message’s.
